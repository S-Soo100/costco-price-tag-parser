# Legacy docs (provenance)

These are the original documents from the **Flutter field-data app** this library
was extracted from. They're kept for provenance — the parsing heuristics and the
46-photo OCR validation that the library is built on were worked out here.

They describe a discarded app (worker/admin flows, a mock backend) and do **not**
reflect the current library. For the library, see the root [README](../../README.md)
and [`spec/`](../../spec).
